# Tolerances

Strict control of agent behavior narrows the mating surface of interacting parts and reduces the tolerances the system needs to absorb imperfection. Decomposition into loosely coupled systems instead lets agents *deduce* correct action without imposed constraints.

The two are easily confused and structurally opposite. Locking down autonomy is micromanagement: every interior decision legislated, judgment replaced with rule-following. Drawing well-defined subdomains — bounded contexts, scoped skills, narrow interfaces — is facilitation: a clearly named remit inside which the agent is free to act. Bounded contexts are facilitation, not control: agents are not prevented from looking outside; the system is designed so they do not need to. Strictness reads as rigor; in practice it produces fragility.
