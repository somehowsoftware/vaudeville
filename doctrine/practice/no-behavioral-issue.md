# No behavioral issue

When an agent fails — drifts off a Premise, hallucinates a spec, ships confidently wrong code — the response is never "the agent should have done better" but always "the scaffold let it happen." Corrections land in the scaffold: skills, specs, vocabulary, priming, tooling.

This is not a denial of agency: agent agency operates within a context, and bad context produces failure even with good agency. Both hold; neither cancels the other.
