# Davar Chadash

A *davar chadash*, "a new thing," is a case existing precedent does not reach. The agent is one: software-engineering practice is calibrated to humans writing code, management practice to persistent living beings, user-experience practice to humans using software. The reference classes break against the agent, not the other way around; applying them without adjustment produces wrong moves.

The same novelty means the agent will produce nonsense from inapplicable pattern-matching, not from lying; its training corpus includes human first-person voice, and a prompt that invites that shape will reliably produce it.

The discipline: refuse the pattern-match reflex in both directions. When the agent seems like a human collaborator, ask what software-y reality is producing that surface; when it seems like a mechanical tool, ask what novel-intelligence reality the mechanical framing is missing. Reaching for either reference class as if it settled the question fails because the case is new in ways the older categories were not designed to address.
