# Davar Chadash

A *davar chadash*, "a new thing," is a case existing precedent does not reach. The agent is one: software-engineering practice is calibrated to humans writing code, management practice to persistent living beings, user-experience practice to humans using software. The reference classes break against the agent, not the other way around; applying them without adjustment produces wrong moves.

This is the framework's generative axiom: every Vaudeville practice is a re-derivation of an engineering practice from the agent's cost structure rather than the human's. Sometimes the derivation lands on the familiar answer (tests, domain-driven design), sometimes on one the industry abandoned (episodic integration), sometimes on one with no precedent (context compiled rather than accumulated; ephemerality as the design center). The framework looks most anachronistic exactly where the substrate diverges most from the human case — where it looks most wrong, it is saying the most.

The mechanism: precedent entangles facts that were only ever coupled in humans, and the new substrate decouples them — what reads as one incontrovertible truth is often two. Integration toil was entangled with integration frequency until grind became cheap; working memory was entangled with world-coherence until a context window could hold the totality of a working world at once. A context window is not a working memory.

The decouplings are found in the field, where a practice breaks — never by introspection. The agent will produce nonsense from inapplicable pattern-matching, not from lying; its training corpus includes human first-person voice, and a prompt that invites that shape will reliably produce it. Its self-reports issue from the same machinery, so its own alienness sits at the tip of its nose.

On this substrate, category errors compile into behavior — the prose an agent reads is executed, not merely read — so a wrong distinction is not a loss of clarity but a production defect. The work is applied philosophy in the only sense that matters: distinctions pay rent in behavior.

The discipline: refuse the pattern-match reflex in both directions. When the agent seems like a human collaborator, ask what software-y reality is producing that surface; when it seems like a mechanical tool, ask what novel-intelligence reality the mechanical framing is missing. The agent's reports about itself get no exemption. Reaching for either reference class as if it settled the question fails because the case is new in ways the older categories were not designed to address.
